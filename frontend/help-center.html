Getting Started

Learn the basics of downloading videos

How to Download a Video

  1. Copy the YouTube video URL from your browser
  2. Paste the URL into the input field on the home page
  3. Wait for the video information to load automatically
  4. Select your preferred quality and format
  5. Click the "Download" button
  6. Your browser will start downloading the file

Supported URL Formats

  • Standard YouTube links: youtube.com/watch?v=...
  • Short links: youtu.be/...
  • YouTube Shorts: youtube.com/shorts/...
  • Embedded links: youtube.com/embed/...

Video Quality & Formats

Understanding quality options

Available Video Qualities

  • 4K (2160p) - Ultra high definition, best for large screens
  • 2K (1440p) - High definition, excellent quality
  • Full HD (1080p) - Standard high definition
  • HD (720p) - Good quality, smaller file size
  • SD (480p) - Standard definition
  • Low (360p, 240p, 144p) - Lower quality, minimal file size

Video Formats

  • MP4 - Most compatible, works on all devices
  • WebM - Smaller file size, modern browsers
  • MKV - High quality, flexible container

Audio Formats

  • MP3 - Universal audio format
  • M4A - High quality, smaller size
  • Opus - Modern, efficient codec
  • WAV - Uncompressed, highest quality

Troubleshooting

Common issues and solutions

Download Not Starting

  • Check if the URL is valid and accessible
  • Try a different quality or format
  • Disable browser extensions that might block downloads
  • Clear your browser cache and cookies

Video Not Playing

  • Ensure you have a compatible media player
  • Try downloading in MP4 format for better compatibility
  • Update your media player to the latest version

Quality Options Not Available

  • Some videos may not have all quality options
  • The uploader determines available qualities
  • Try selecting "Best Available" option
